# Advanced HTML

Structure of web pages using semantic HTML5 tags. No CSS — focus is purely on markup.

**Company used throughout the project:** Techium

## Learning Objectives

- Create the skeleton of an HTML5 page
- Use semantic tags: `header`, `main`, `footer`, `article`, `nav`, `section`, `aside`
- Difference between `div` and `span`
- Use headings in hierarchical order
- Create lists, tables, embed video/audio/external content
- Differences between image formats (SVG, GIF, PNG, JPG)

## Requirements

- Code must be W3C compliant
- `README.md` mandatory at root of project
